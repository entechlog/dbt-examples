üöÄ Quick Start

1. Install Dependencies

# Install dbt packages
dbt deps

# Install dbt-model-erd
pip install dbt-model-erd

2. Load Data and Run Models

# Load seed data
dbt seed

# Run models
dbt run

3. Generate ERD Diagrams

# Generate ERDs for fact models
python -m dbt_erd --model-path models/dw/fact

# View generated diagrams
open assets/img/models/dw/fact/fact_orders_model.html
